# Mesh Assets

These assets are used by the tutorial as external mesh inputs rather than handwritten arrays.

- `stanford_bunny.obj`: downloaded from `https://graphics.stanford.edu/~mdfisher/Data/Meshes/bunny.obj`. The Stanford Bunny originates from the Stanford 3D Scanning Repository / Stanford Bunny model.
- `cornell_box.stl`: downloaded from Wikimedia Commons file `Cornell Box (flipped) and camera parameters.stl`.
- `cornell_box.obj`: converted locally from `cornell_box.stl` for the tutorial OBJ loader.

The tutorial keeps these as input assets so readers can replace them with their own mesh files.
