# Diagram Assets

这些图解用于网页书正文。

优先用 GPT Image 生成有教学质感的位图图解；如果某张图需要严格对齐、可编辑文字或可维护的结构，也可以用 SVG 手写。当前 SVG 是可立即使用的确定性占位图。

计划用 GPT Image 生成或重绘的图：

- `device-command-queue.png`: Device、Command Queue、Command Buffer 的关系。
- `buffer-round-trip.png`: CPU buffer 到 GPU buffer 的往返。
- `pixels-grid.png`: 图片如何变成 RGB 像素网格。
- `compute-grid.png`: grid、threadgroup、thread 与像素的关系。
- `blur-kernel.png`: 3x3 blur kernel 的邻域采样。
- `compute-vs-render.png`: compute pipeline 和 render pipeline 的对比。
