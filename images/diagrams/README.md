# Diagram Assets

These illustrations are used in the text of the web book.

Prioritize using GPT Image to generate bitmap illustrations with a teaching quality; if a certain image requires strict alignment, editable text, or maintainable structure, you can also use hand-authored SVG. The current SVG files are deterministic placeholder diagrams that can be used immediately.

Plan to generate or redraw images using GPT Image:

- `device-command-queue.png`: Relationship between Device, Command Queue, and Command Buffer.
- `buffer-round-trip.png`: Round trip from CPU buffer to GPU buffer.
- `pixels-grid.png`: How to turn an image into an RGB pixel grid.
- `compute-grid.png`: The relationship between grid, threadgroup, thread and pixel.
- `blur-kernel.png`: Neighborhood sampling of 3x3 blur kernel.
- `compute-vs-render.png`: Comparison between compute pipeline and render pipeline.
